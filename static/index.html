About this app
Powered by OpenAI Whisper (faster-whisper), an open-source speech recognition system developed by OpenAI. Runs entirely on your computer — no audio is ever sent to any external server or cloud service. Transcripts are saved locally as .txt files in the transcriptions/ folder.
